Plataforma Cívica: "Gallardo te Escucha"

Sistema modular de seguridad e infraestructura urbana para la gestión municipal.

"Gallardo te Escucha" (Plataforma Cívica Modular V1) es una aplicación web enfocada en mejorar la conexión entre los ciudadanos y las autoridades de Villa Santa Cruz del Lago. Nace como una herramienta estratégica para digitalizar y centralizar reportes, permitiendo armar un mapa de inteligencia sobre problemáticas barriales en tiempo real.

Arquitectura de Roles y Accesos

El sistema cuenta con tres niveles de interactuantes explícitos para garantizar la seguridad y veracidad de la información:

  • El Vecino (Ciudadano): Acceso público validado por DNI (modelo anti-spam). Permite reportar delitos, accidentes o problemas urbanos adjuntando ubicación exacta y evidencia fotográfica.
  • Autoridad (Policía / Mantenimiento): Acceso protegido mediante credenciales. Permite visualizar un Mapa de Inteligencia en tiempo real para consumir los incidentes verificados y planificar patrullajes o reparaciones.
  • Super Administrador: Acceso restringido para tareas de triaje. Modera y aprueba los reportes ingresados antes de su publicación en el mapa general, evitando la difusión de datos falsos o duplicados.

Diferenciadores Estratégicos y Tecnología

Georeferenciación Integración de mapas interactivos (Leaflet) para la ubicación precisa de incidentes mediante pines dinámicos.
Gestión de Evidencia Las imágenes se procesan en Base64, archivándose como código para eficientizar el alojamiento web sin depender de servidores de almacenamiento costosos.
Accesibilidad (PWA) Desarrollada para operar completamente desde un enlace web. No requiere descargas desde tiendas de aplicaciones, permitiendo su distribución viral y directa vía WhatsApp.
Filtros Analíticos El mapa oficial cuenta con filtros temporales (24hs, semanal, mensual) para el análisis estadístico de las fuerzas de seguridad.

* Nota: El sistema prevé futuras integraciones mediante WebSockets y Chatbots para alertas instantáneas de emergencia (Comando C4).
